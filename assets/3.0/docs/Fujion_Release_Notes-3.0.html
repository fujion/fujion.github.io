Fujion Release Notes

Version 2.2.0

29-Aug-2018

  • Component updates
    • New constructors for Listitem, Comboitem, and Groupbox
    • Listitem and Comboitem now implement Comparable interface
    • Refactoring of component tracking mechanism
    • Hover popups no longer flicker
    • Hint popups now use bootstrap popover styling
    • Hint popups now use hover popup logic
    • New global properties to control hover popup delay and duration
    • Fixed hover popup issues with Treenode components
  • Core updates
    • New ConvertUtil.getAttributeAs methods
    • New MiscUtil.castCollection method
    • Added support for global client settings
    • Added support for serialization of custom datatypes
  • Library updates
    • Moved to Fujion-managed webjars
    • Feather Icons version update to 4.7.3
    • Groovy version update to 2.5.2
    • Renjin version update to 0.9.2669
    • CodeMirror version update to 5.4.0
    • Plotly version update to 1.40.1
    • Angular version update to 6.1.4

Version 2.2.1

19-Sep-2018

  • Component updates
    • Javascript property setters must now begin with s_ prefix

Version 3.3.0

29-Aug-2020

  • Component updates
    • New global properties for controlling hint behavior
    • Button component now supports top and bottom label alignment
    • Checkbox component now supports top and bottom label alignment
    • Can disable hint tooltips globally via Page component
    • New Page.requestUrl convenience method
    • New markup component
    • New mxGraph component
    • Localization support for Paging navigation labels
  • Core updates
    • New thread pool factory
    • Page elements support optional constructor arguments
    • Websocket session object now exposes properties for application context and thread pool
  • Library updates
    • Version updates for all dependencies
    • Move to new WebJar format using embedded config file
    • Collection methods in MiscUtil moved to new CollectionUtil class
