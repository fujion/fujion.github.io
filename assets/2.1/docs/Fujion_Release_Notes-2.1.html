Fujion Release Notes

Version 2.1.0

9-Aug-2018

  • New components
    • Module - for on-demand asynchronous loading of JavaScript modules
    • CodeMirrorXML - CodeMirror component extension optimized for editing XML
  • Component updates
    • More configuration options for CodeMirror component
    • Barcode component now supports QR codes
    • New switch/case component factory parameters
  • Core updates
    • New BaseComponent.loadModule method
    • New BaseComponent.detachChildren method
    • BaseComponent.getController renamed to getLastController
    • BaseComponent.wireController now accepts Collection data type
    • New mode attribute on @WiredComponent and @EventHandler annotations for more fine-grained control over wiring controllers
    • New PageParser.registerAttributeNS method for registering attribute namespace extensions
    • List returned by BaseComponent.getChildren now mutable
    • New MiscUtil.globToRegex method
    • New IAttributeMap interface
    • New MessageBundle class
    • New extensible JMX-based management console
  • Client updates
    • Support for callbacks on async client invocations
  • Sandbox updates
    • Context-aware content completion
    • New FSP skeleton template
  • Library updates
    • Spring Framework version update to 5.0.8
    • Angular version update to 5.2.10
    • React Framework version update to 16.4.2
    • Version updates for most dependent libraries
